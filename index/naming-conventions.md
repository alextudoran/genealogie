# Convenții de numire

## Fișiere pentru persoane

Format general:

```text
people/<familie>/<prenume-nume-suplimentar>.md
```

Exemple:

```text
people/lacanu/tudor-t-lacanu.md
people/lacanu/mihai-st-lacanu.md
people/unknown/unknown-lacanu-male-1908-01.md
```

## Fișiere pentru familii

Format:

```text
families/<nume-familie>.md
```

Exemplu:

```text
families/lacanu.md
```

## Documente

Documentele trebuie numite după cotă, tip, an sau conținut identificabil.

Exemple:

```text
documents/archive-files/vn-f-00021-3-1908-153.md
documents/civil-records/1908-vulturu-casatorie.md
```

## Cote arhivistice

Cota arhivistică primește fișier propriu în `archive-references/`.

Exemplu:

```text
archive-references/vn-f-00021-3-1908-153.md
```

## Ipoteze

Format:

```text
hypotheses/<subiect-ipoteza>.md
```

Exemple:

```text
hypotheses/st-lacanu-patronymic-analysis.md
hypotheses/stoica-patronymic.md
```

## Reguli generale

- Folosește litere mici.
- Folosește cratimă, nu spații.
- Păstrează diacriticele în conținut, nu neapărat în numele fișierului.
- Nu redenumi fișiere fără motiv; linkurile interne se pot rupe.
