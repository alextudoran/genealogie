# Genealogie

Repository pentru cercetare genealogică, istorie locală și documentarea surselor legate de familii, persoane, localități și documente de arhivă.

## Principii

- Documentele sunt sursa; persoanele sunt interpretarea.
- Faptele confirmate se separă de ipoteze.
- Orice afirmație genealogică trebuie legată de o sursă.
- Nu forțăm identificări. Persoanele necunoscute pot rămâne necunoscute.
- Contextul local contează: martori, nași, vecini, proprietăți, cote arhivistice și localități pot deveni dovezi.

## Structură

- `index/` — puncte de intrare și liste de control.
- `people/` — fișiere pentru persoane individuale.
- `families/` — pagini pentru familii și nume de familie.
- `places/` — localități, regiuni, zone istorice.
- `documents/` — documente, transcrieri și extrase.
- `archive-references/` — cote arhivistice și descrieri de dosare.
- `hypotheses/` — ipoteze active și niveluri de încredere.
- `timelines/` — cronologii familiale, locale și personale.
- `research-log/` — jurnal cronologic al cercetării.
- `templates/` — șabloane de documentare.

## Punct de pornire

Începe cu `index/master-index.md`, apoi verifică regulile din `index/documentation-rules.md`.
