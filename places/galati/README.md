# Galați

## Status

Localitate de tranzit importantă pentru ramura Tudoran.

## Relevanță genealogică

Conform memoriei familiale, străbunica Tudoran, născută la Ismail la 28 noiembrie 1917, a plecat la aproximativ 16 ani la Galați.

La Galați:

- a lucrat ca servitoare / fată în casă;
- ar fi stat aproximativ doi ani;
- conform relatării bunicii cercetătorului, proprietarul / boierul la care lucra ar fi profitat de ea;
- după această etapă a plecat la Constanța împreună cu copilul ei, Aurel Tudoran.

## Observații metodologice

- Informația despre abuz este tradiție orală și trebuie păstrată ca atare până la apariția unei surse documentare.
- Galați poate fi relevant pentru găsirea primelor documente de domiciliu / angajare / naștere / botez, dar șansele sunt incerte.
- Dacă Aurel Tudoran s-a născut în Galați sau imediat după plecarea din Galați, actul de naștere complet poate clarifica etapa.
