# Vulturu

## Identificare

- Localitate principală: **Vulturu**
- Județ modern: **Vrancea**
- Context istoric principal: fost județ **Putna**, plasa **Biliești / Biliesci**
- Formă istorică frecventă în surse: **Vulturul**

## Variante de notare urmărite

### Variante confirmate / folosite în surse

- **Vulturu** — forma modernă.
- **Vulturul** — formă istorică frecventă în surse vechi.
- **Vulturu de Jos** / **Vulturul de Jos** / **Vulturul-de-Jos** — fostă componentă / formulă administrativă.
- **Vulturu de Sus** / **Vulturul de Sus** / **Vulturul-de-Sus** — fostă componentă / formulă administrativă.
- **Vulturul Nou** / **Sloboziea Vulturul Nou** — formulă istorică de urmărit în catagrafii / recensăminte vechi.
- **Biliești** / **Biliesci** — plasa relevantă pentru localizarea Vulturului în surse din secolul al XIX-lea.

### Variante de căutare de testat / neconfirmate încă în documentele salvate

- **Hultur**
- **Hulturul**
- **Hulturu**
- **Hulturulu**

Notă: variantele cu `H` sunt păstrate ca ipoteze de căutare / OCR / transliterare. Nu se tratează ca forme documentate până când apar într-o sursă verificată.

## Sate / localități imediat asociate cu cercetarea Vulturu

### Componente sau localități foarte apropiate

- **Boțârlău** / **Botârlău** / **Botîrlău** — sat asociat cu Vulturu; apare în documente civile ca domiciliu / loc de legătură pentru ramuri locale.
- **Vadu Roșca** / **Vadul Roșca** / **Vadul-Roșca** / **Vadu-Roșu** — localitate / formulă apropiată; apare în contextul Vulturu, hărți și istorie locală.
- **Maluri** / **Malurile** / **Marlurile** / **Sloboziea Nouă Malurile** — localitate apropiată și relevantă pentru acte civile / conexiuni familiale.
- **Hângulești** / **Hîngulești** — localitate devenită relevantă pentru ramuri Lăcanu și pentru registrele civile conexe.
- **Gulianca** — localitate / zonă urmărită în clusterul Hângulești / Gulianca / Măicănești.
- **Măicănești** — localitate urmărită în clusterul Hângulești / Gulianca / Măicănești și în documente civile conexe.

### Localități de vecinătate / context regional apropiat

- **Suraia** — apare în acte civile legate de Vulturu; relevantă pentru conexiuni matrimoniale.
- **Gologanu** — apare în acte civile ca loc de naștere pentru persoane căsătorite în Vulturu.
- **Călieni** — localitate din zona Biliești / Putna; relevantă pentru context administrativ și agrar.
- **Răstoaca** — localitate vecină / de context în zona Putna.
- **Biliești** — centru de plasă / reper administrativ, nu neapărat localitate genealogică directă pentru familie.
- **Soci** — relevant mai ales pentru context istoric-arheologic / medieval, nu pentru ramuri familiale directe încă.

### Toponime / zone locale non-sat

- **Gârlița** — zonă menționată în amintiri familiale; posibil braț secat al Putnei sau alt curs de apă.
- **Putna** — râu și fost județ; în cercetare trebuie clarificat de fiecare dată dacă se referă la râu, județ sau context administrativ.
- **Siret** — reper hidrografic major în zona Vulturu.
- **Drumul / șoseaua Brăila–Focșani** — reper de graniță / circulație locală.
- **Cotul moldovenesc** și **cotul muntenesc** — repere din memorie locală; de clarificat ca localizare exactă.
- **Pichete de grăniceri** — reper de memorie locală legat de fostul hotar.

## Relevanță genealogică

Localitate centrală pentru cercetarea familiei **Lăcanu** și a contextului rural local. Prin actele civile și sursele istorice conexe, Vulturu este legată și de numele / familiile **Miron / Mironescu**, **Mocăniță**, **Vasile** și **Leoca / Leocă**.
