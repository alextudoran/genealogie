# Persoane necunoscute

Acest director este pentru persoane menționate în documente, dar care nu pot fi încă identificate sigur.

## Reguli

- Nu forța identificarea unei persoane necunoscute.
- Creează un fișier temporar când o persoană apare suficient de importantă pentru cercetare.
- Leagă fiecare persoană necunoscută de documentul în care apare.

## Convenție nume fișier

```text
unknown-[familie-dacă-există]-[localitate]-[an]-[număr].md
```

Exemplu:

```text
unknown-lacanu-vulturu-1908-01.md
```
